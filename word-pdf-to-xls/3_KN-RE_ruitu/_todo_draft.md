### 1. 提取词汇 → 执行
python extract_vocab.py
生成 `result_phase12.xls`

### 2. 手工去重
用Excel打开"3去重后"Sheet，合并重复词汇，保存
  - 黄色 (light_yellow) — 重复词汇的首次出现
  - 橙色 (light_orange) — 重复词汇的第二次及后续出现
  - 蓝色 (light_blue) — 中文释义超过 50 个字符（过长）

### 3. 分类导出 → 执行
python split_upload.py
生成 `result_phase4.xls`，含"5拟上传"和"5不上传"两个Sheet
  1. 只放拟上传 — 普通词条，没有特殊标记，只进"5拟上传"sheet
  2. 只放不上传 — 英文词汇含 ' 或 ... 的词条（如 it's、etc...），只进"5不上传"sheet
  3. 两边都放 — 蓝色背景的词条（中文释义超长），同时放进两个 sheet
