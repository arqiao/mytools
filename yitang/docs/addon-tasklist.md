## 阶段一：基础框架与配置

### 1.1 配置文件设计 (addon.yaml)
- ✅ 1.1.1 定义 addon.yaml 结构：逐字稿本地文件路径、字幕文件路径、讨论区文件路径
- ✅ 1.1.2 定义 LLM 配置项：支持 minimax/volcengine/dashscope/deepseek 四家切换
- ✅ 1.1.3 定义输出配置：输出目录、文件命名规则
- ✅ 1.1.4 在 credentials.yaml 中添加 LLM API key 字段（minimax/volcengine/dashscope/deepseek）

### 1.2 程序骨架
- ✅ 1.2.1 创建 `src/yitang_addon.py` 主程序框架：参数解析、配置加载、日志初始化
- ✅ 1.2.2 实现文件读取模块：读取逐字稿(md/txt)、字幕(.srt)、讨论区(.xlsx)
- ✅ 1.2.3 实现 SRT 字幕解析：提取纯文本内容，保留时间标记便于溯源
- ✅ 1.2.4 实现讨论区 Excel 解析：提取发言人、时间、内容
- ✅ 1.2.5 实现逐字稿章节结构解析：识别标题层级，拆分为章节段落

### 1.3 逐字稿输入源适配
- ✅ 1.3.1 addon: transcript 支持飞书 URL 输入，自动通过飞书 API 拉取文档内容并解析章节
- ✅ 1.3.2 addon: transcript 支持本地文件路径输入，自动判断 URL 还是本地文件
- ✅ 1.3.3 wiki: config-wiki.yaml 增加 `local_export` 配置项（每个 mapping 可选）
- ✅ 1.3.4 wiki: yitang_wiki.py 新增 `export_local_md()` 方法，写入飞书后同步导出本地 md

- ✅ 2.2.3 设计"章节归类"提示词：将提取结果按逐字稿章节分类
- ✅ 2.2.4 提示词中明确要求详细输出（对抗 LLM 默认的简洁倾向）
